refleja mi proceso de aprendizaje y evolución como desarrolladora web.

Mi WEBnezuela es una página informativa que resalta la belleza natural, histórica y cultural del parque nacional El Ávila (Waraira Repano). El sitio incluye secciones como historia, rutas, galería de imágenes y datos de contacto, todo desarrollado con HTML, CSS, Bootstrap y JavaScript. Este proyecto fue una práctica integral para aplicar diseño responsivo y estructuración de contenido en una web estática.

Té y un libro es una página dedicada a compartir el placer de la lectura. En ella se exploran anécdotas, curiosidades y reflexiones sobre libros, creando un espacio cálido e inspirador para lectores. El diseño sencillo y acogedor fue construido con HTML y CSS, priorizando la claridad del contenido y la comodidad visual.

Reflexiones es una página centrada en el autoconocimiento y el bienestar emocional, inspirada en principios de la psicología positiva. A través de textos breves y mensajes diarios, invita a explorar aspectos personales como la gratitud, la calma, y el crecimiento interior. El diseño busca ser sereno y minimalista, utilizando HTML, CSS y sonidos relajantes para crear un ambiente introspectivo.

Conocimientos Técnicos: Cuento con formación en desarrollo web, trabajando con tecnologías como HTML, CSS y JavaScript para la creación de páginas responsivas y funcionales. También tengo conocimientos en Python para automatización y lógica de programación, así como conocimientos en Prompt Engineering para el diseño efectivo de instrucciones en entornos de inteligencia artificial. Además, manejo la instalación y configuración de sistemas operativos, lo que complementa mi perfil técnico y me permite abordar proyectos desde una perspectiva integral.
